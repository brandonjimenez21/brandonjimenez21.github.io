> Portafolio web moderno, minimalista y completamente responsive con características avanzadas. Creado con HTML5, CSS3 y JavaScript vanilla (sin frameworks).

---

## ✨ Características Principales

### 🎨 Diseño y Estilo
- **Minimalista y profesional** con paleta de colores moderna (Índigo, Rosa, Turquesa)
- **100% Responsive**: Móvil, tablet y desktop
- **Tipografía profesional**: Inter (body) + Poppins (headings)
- **Sistema de variables CSS** para fácil personalización
- **Animaciones fluidas**: Fade in, slide, typing effect, parallax

### 🌙 Características Avanzadas
- **Modo Oscuro (Dark Mode)**: Toggle con persistencia en localStorage
- **Bilingüe (ES/EN)**: Cambio de idioma completo sin recargar
- **Preloader Minimalista**: Animación de código escribiéndose
- **Cursor Personalizado**: Efecto profesional en desktop
- **Glassmorphism**: Efectos de vidrio esmerilado en tarjetas
- **Partículas de Fondo**: Animación sutil con Canvas API
- **Easter Egg**: Sorpresa con CTRL+SHIFT+F

### 📊 Secciones Implementadas
1. **Hero/Inicio**: Presentación con efecto de escritura automática
2. **Sobre Mí**: Información personal y profesional con estadísticas
3. **Habilidades**: Tecnologías con porcentajes interactivos tooltip
4. **Estadísticas Animadas**: Contadores con scroll trigger
5. **Proyectos Destacados**: Portfolio con enlaces y tecnologías
7. **Hobbies**: Intereses personales con iconos
8. **Contacto**: Formulario funcional con FormSubmit + redes sociales
9. **Footer**: Navegación, botón de compartir y contador de visitas

### 🔧 Funcionalidades Técnicas
- **Smooth Scroll Navigation**: Navegación suave entre secciones
- **Active Link Detection**: Resalta sección actual en navbar
- **Form Validation**: Validación en tiempo real del formulario
- **LocalStorage System**: Guarda tema, idioma, comentarios y visitas
- **Menú Hamburguesa**: Navegación móvil completamente funcional
- **Share Buttons**: Compartir en Twitter, Facebook, LinkedIn, WhatsApp

---

## 🚀 Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible
- **CSS3**: Variables CSS, Grid, Flexbox, Animaciones, Backdrop-filter
- **JavaScript ES6+**: Vanilla JS sin dependencias
- **Font Awesome 6.5.1**: Iconos profesionales
- **Google Fonts**: Inter y Poppins
- **FormSubmit.co**: Servicio de formularios sin backend
- **Canvas API**: Preloader y partículas animadas
